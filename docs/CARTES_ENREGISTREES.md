# Cartes enregistrées

## Carte dynamique

La carte est reconstruite depuis le référentiel à chaque ouverture. Elle convient aux vues opérationnelles permanentes.

## Instantané

Le résultat et l’état ouvert des bulles sont conservés lors de l’enregistrement. Il convient aux audits, dossiers de preuve et comparaisons avant/après.

## Constructeur pleine page

Depuis **Cartes enregistrées → Construire une carte**, l’utilisateur prépare la sélection sur une page complète, l’enregistre, puis ouvre automatiquement sa visualisation.

Le constructeur couvre les objets de départ, les types, les relations, le sens, la profondeur, les filtres, les niveaux imbriqués, les regroupements par étiquette ou information complémentaire, la visibilité et la protection.

Le panneau **Sélection** demeure disponible dans la carte pour modifier une configuration existante.

## Création et remplacement

Deux possibilités sont proposées lors de l’enregistrement :

- créer une nouvelle carte ;
- sélectionner explicitement une carte existante modifiable et la remplacer.

Le remplacement demande une confirmation. Le bouton **Mettre à jour** remplace directement les paramètres de la carte active.

## Paramètres enregistrés

- objets de départ ;
- types d’objets et de relations ;
- profondeur et direction ;
- filtres ;
- niveaux de granularité ;
- regroupements par étiquette ou information complémentaire ;
- disposition, positions et caméra ;
- visibilité et groupes autorisés ;
- mention de protection.

## Exports

Depuis la visualisation :

- PDF A3 paysage multipage avec identité visuelle, légendes et inventaires détaillés des objets et relations ;
- PNG haute définition ;
- JSON de configuration pour une carte enregistrée.

L’API conserve également les exports JSON, CSV objets et CSV relations.
