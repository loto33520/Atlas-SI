# Constructeur de cartes enregistrées

## Principe

La page **Cartes enregistrées** comprend un constructeur en pleine page. Le parcours conseillé est :

1. préparer la sélection et le mode d’exploration ;
2. enregistrer la carte ;
3. ouvrir automatiquement sa visualisation ;
4. ajuster ensuite la sélection ou la disposition directement dans la carte si nécessaire.

## Création ou remplacement

Le champ **Destination** permet de créer une nouvelle carte ou de choisir une carte existante modifiable. Un remplacement nécessite une confirmation.

## Paramètres disponibles

- nom, description, visibilité et mention de protection ;
- objets de départ avec recherche ;
- direction amont, aval ou bidirectionnelle ;
- carte standard, exploration progressive ou exploration imbriquée ;
- profondeur pour une carte standard ;
- types d’objets et de relations avec recherche ;
- niveaux de granularité réordonnables ;
- regroupement d’un niveau par étiquette ou information complémentaire ;
- criticité et filtres d’étiquettes ;
- disposition initiale.

Une carte construite sur cette page est enregistrée en mode dynamique. Elle peut ensuite être enregistrée comme instantané depuis la visualisation.
